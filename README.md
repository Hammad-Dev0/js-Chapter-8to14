# JavaScript Foundations: Chapters 1–14 🚀

A professional interactive web application built to demonstrate core JavaScript concepts, including data types, arithmetic operations, conditional logic, and DOM manipulation. This project was developed as part of the **SMIT (Saylani Mass IT Training)** WMA 2025-26 session.

## 🌟 Key Features
- **Interactive UI:** A clean, "boutique-style" interface using Tailwind CSS.
- **Dynamic Feedback:** Integration of **SweetAlert2** for professional modal popups and **Toastify JS** for non-intrusive notifications.
- **Comprehensive Logic:** Covers foundational JS topics from basic alerts to complex nested conditional statements.
- **Responsive Design:** Fully optimized for mobile, tablet, and desktop views.

## 🛠️ Technologies Used
*   **HTML5 & CSS3:** For semantic structure and layout.
*   **Tailwind CSS:** Modern utility-first styling for a high-end look.
*   **JavaScript (ES6+):** Core logic, event handling, and DOM manipulation.
*   **Libraries:** 
    *   [SweetAlert2](https://sweetalert2.github.io/) (Modals)
    *   [Toastify JS](https://apvarun.github.io/toastify-js/) (Toasts)

## 📚 Concepts Covered
The application provides interactive examples for:
1.  **Alerts & Data Types:** Displaying names and numbers.
2.  **Naming Conventions:** Variable naming rules and CamelCase examples.
3.  **Arithmetic Operators:** Addition, Subtraction, Multiplication, Division, and Complex Calculations.
4.  **String Manipulation:** Concatenation using Template Literals.
5.  **User Input:** Prompt handling and validation.
6.  **Conditional Logic:** 
    *   Comparison Operators
    *   `if...else if...else` structures
    *   Testing multiple sets of conditions (`&&` / `||`)
    *   Nested `if` statements
7.  **Functional Logic:** Basic Login authentication and Grade checking system.
